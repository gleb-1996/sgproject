О нас

Сегодня ООО «Стройгазпроект» — это успешная компания с многолетним опытом работы, положительной деловой репутацией, ориентированная на построение взаимовыгодных и долгосрочных отношений со своими заказчиками.

Семинары

Приобретение слушателями теоретических знаний по реализации единого подхода к оформлению исполнительной документации, а также практических навыков по оформлению исполнительной документации по специализациям.

Проекты

Разработка разделов проектной документации «Перечень мероприятий по гражданской обороне, мероприятий по предупреждению чрезвычайных ситуаций природного и техногенного характера», «Мероприятия по обеспечению пожарной безопасности», «Перечень мероприятий по охране окружающей среды» по объектам реконструкции (ОАО «ГИПРОИВ», 2014-2015)
