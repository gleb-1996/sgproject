Инженерные изыскания

Корректировка документации по планировке территории в рамках выполнения комплекса работ по оформлению прав на земельные участки для эксплуатации объектов в составе инвестиционного проекта «Магистральный газопровод «Моздок – Грозный» путем установления публичного сервитута (ООО «КЭТ», 2022).

Выполнение подготовительных работ по оформлению прав ПАО «Газпром» на земельные (лесные) участки под объект «Реконструкция магистрального газопровода ШБКБ на участке 120-160 км» (ООО «КЭТ», 2021).

Выполнение подготовительных работ по установлению (формированию) границ земельных участков под объект «Оснащение ИТСО объектов КС Таежная газопровода Ямбург – Тула 1» (ООО «КЭТ», 2021).

Инженерные изыскания и получение Градостроительного плана земельного участка по объекту «Воронежская область, г. Бобров, АГНКС-1» (ООО «Газпром проектирование», 2019).

Сбор исходных данных для подготовки документации по планировке территории по объекту «Развитие газотранспортных мощностей ЕСГ Северо-Западного региона, участок Грязовец - КС Славянская» (ООО «Газпром проектирование», 2017).

Натурное обследование по выявлению объектов историко-культурного наследия по объекту ГРС Лаголово (АО «Гипроспецгаз», 2016).

Разработка и утверждение проектов рекультивации земельных участков под скважинами (ООО «Газпром добыча Уренгой», 2015-2016).

Обследование фундаментов на объекте реконструкции в г.Реутов (АО «ВПК «НПО машиностроения», 2015).

Разработка и согласование проектов рекультивации нарушенных земель при строительстве 12 линейных и площадных объектов стройки «Система магистральных газопроводов Бованенково-Ухта» (ЗАО «Ямалгазинвест», 2014).

Выполнение комплекса работ по созданию геодезической разбивочной основы 3 линейных участков стройки «Система магистральных газопроводов Бованенково-Ухта» (ЗАО «Ямалгазинвест», 2011).

Выполнение комплекса работ по созданию картографических схем земельных и лесных участков, объектов строительства линейной части стройки «Газопровод Починки – Грязовец» (ЗАО «Ямалгазинвест», 2011).
